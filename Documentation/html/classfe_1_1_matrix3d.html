Detailed Description

template<typename T>
class fe::Matrix3d< T >

This class is a template class that offers simple 3D arrays manipulations.

Matrix3d is used to handle 3D arrays of any type. It emulates 1D array into 3D one. For the user it works like a std::vector, but with 3D functionalities.
